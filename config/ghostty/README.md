To fix the cmd+c keybind:

1. **System Settings → Keyboard → Keyboard Shortcuts… → App Shortcuts**
2. Click **+**
3. **Application:** `supacode.app`
4. **Menu Title:** `Copy`  *(must match the Edit-menu item exactly)*
5. **Keyboard Shortcut:** press an alternative, e.g. **⌘^C**
6. **Add**, then fully quit and relaunch Supacode.
